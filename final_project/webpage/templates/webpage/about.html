Our CPEN 291 Final Project

People enjoy listening to music during different times of the day and accordingly, choose specific genres of music based on their current mood and feelings. This project focusses on building a web application that predicts users' moods (e.g. sadness, anger, happiness, neutral) based on their facial expressions using computer vision. Based on the prediction of our machine learning model, the app outputs a Spotify playlist that suits the result.

Our backend was built using Django framework and Python. Our machine learning model was built using Pytorch together with a dataset comprising of 500-600 images for each emotion. The web frontent was designed using HTML, CSS, Javascript and Bootstrap.
